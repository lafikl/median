Median
=========

A small module for node and browser to calculate median.


## Usage

**Node.js**
```js
var median = require('median')
var arr = [1, 15, 200, 30]
var value = median(arr)

console.log("median value is " + value)
```
 
 


**Browser**
```js
var arr = [1, 15, 200, 30]
var value = window.median(arr)

console.log("median value is " + value)
```
